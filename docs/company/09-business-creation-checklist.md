## PHASE 1 — INCORPORATION (JOUR 1-2)

### □ 1. Choisir la forme juridique
```
Recommandé : Société par actions provinciale (Québec)
  + Plus simple et moins cher que fédérale
  + Données via le Registraire des entreprises QC
```

### □ 2. Réserver et enregistrer le nom
```
→ Vérifier disponibilité : https://www.registreentreprises.gouv.qc.ca
→ "Comptaria" — vérifier disponibilité
→ Nom à dépôt (avec numéro) : inclus dans l'incorporation
```

### □ 3. S'incorporer
```
Outil en ligne : https://www.registreentreprises.gouv.qc.ca
  → "Déposer une demande d'immatriculation"
  → Frais : 332$

Ce que tu obtiens :
  ✓ Numéro d'entreprise du Québec (NEQ)
  ✓ Numéro d'immatriculation
  ✓ Personnalité juridique distincte (responsabilité limitée)
```

### □ 4. Registre fédéral (optionnel mais recommandé)
```
Si tu veux opérer hors Québec un jour :
  → Incorporation fédérale : https://www.ic.gc.ca
  → Coût : 200$ fédéral + 177$ QC = 377$
  → Avantage : protection du nom dans tout le Canada
```

```

```

→ https://www.revenuquebec.ca
→ Coût : Gratuit — Délai : Immédiat
```

### □ 7. Numéros de programme pour retenues sur la paie
```
Si tu prévois embaucher :

  → Coût : Gratuit — Prends-les maintenant

```
À préparer AVANT le premier client :
  → Template de contrat de service IA compta
  → Clause de confidentialité (NDA)
  → Clause de propriété intellectuelle
  → Clause de limitation de responsabilité
  → Modalités de paiement (50% upfront, 50% livraison)

```
